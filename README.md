# Meta Ad Performance Dashboard | Power BI

## 📊 Project Overview
Analyzed 10K+ ad impressions to track CTR, conversion rate, and engagement across Facebook & Instagram using Power BI.

## 🎯 Key Insights
- **13.50% Engagement Rate** with 5.98% Conversion Rate
- **CTR: 11.52%** - 3x industry average
- **Top Age Group:** 20-30 years drives max engagement
- **Total Budget Optimized:** $2.54M across campaigns

## 🛠️ Tools Used
Power BI, DAX, Data Modeling, Data Visualization
